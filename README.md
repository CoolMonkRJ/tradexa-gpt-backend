# 📈 Tradexa GPT

A Spring Boot backend project that analyzes trading journals and provides trading analytics.

This project is being built from scratch following production-level backend practices.

---

## 🚀 Tech Stack

- Java 21
- Spring Boot
- Maven
- REST APIs
- In-Memory Storage (Current Phase)

---

## ✅ Features Completed

### CRUD APIs

- Create Trade
- Get All Trades
- Get Trade By ID
- Update Trade
- Delete Trade

### Backend Concepts

- Layered Architecture
- REST API Design
- ResponseEntity
- Generic API Response Wrapper
- Constructor Dependency Injection
- Exception Handling
- Global Exception Handler (@ControllerAdvice)
